# Getting Started

WalletConnect is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking.

This page will help you get started with WalletConnect and the various SDKs we provide.

<CloudBanner />

### Web3Modal

:::tip When should I use this?
Web3Modal is best suited for dapp developers
:::

The Web3Modal SDK allows you to easily connect your Web3 dapp with wallets. It provides a simple and intuitive interface for dapps to request actions such as signing transactions and interacting with smart contracts on the blockchain.

### Web3Wallet

:::tip When should I use this?
Web3Wallet is best suited for wallet developers
:::

The Web3Wallet SDK allows you to easily connect your Web3 wallet with dapps. It provides a simple and intuitive interface for dapps to request actions such as signing transactions and interacting with smart contracts on the blockchain.

### Web3Inbox

:::tip When should I use this?
Web3Inbox is best suited for chat/messaging/notification developers
:::

The Web3Inbox SDK allows one to easily make use of WalletConnect's [notify](https://specs.walletconnect.com/2.0/specs/clients/notify/client-api) and [chat](https://specs.walletconnect.com/2.0//specs/clients/chat/client-api.md) SDKs.

### WalletConnectModal

:::tip When should I use this?
WalletConnectModal is useful for developers building on native platforms. It is also best suited for dapp developers who want granular control over the modal.
:::

The WalletConnectModal SDK allows you to get granular control over how WalletConnect's Modal works. You can think of it as a lightweight alternative to Web3Modal.

# React Introduction

Web3Modal provides first class support for React framework.
To further accelerate your experience it integrates seamlesly with [wagmi](https://wagmi.sh/react/getting-started) on EVM chains.
For more advanced multichain use cases Web3Modal can always be used with [WalletConnect Sign API](../../../api/sign) or [WalletConnect Auth API](../../../api/auth).

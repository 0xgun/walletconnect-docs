# HTML Introduction

Web3Modal is built on top of battle tested web standards like web animations api and web-components, so it can be used in any html environment.
To further accelerate your experience it integrates seamlesly with [wagmi](https://wagmi.sh/core/getting-started) on EVM chains.
For more advanced multichain use cases Web3Modal can always be used with [WalletConnect Sign API](../../../api/sign) or [WalletConnect Auth API](../../../api/auth).
